# Fedora Post Install

This git repo consists of Fedora (27) system/software configuration files and install scripts.

## Getting Started

Post install of a Fedora system, clone to user home directory and run scripts.

## Prerequisites

Internet + wget

## Installing

Clone it! Run scripts!
